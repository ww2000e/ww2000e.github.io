BeeGFS

BeeGFS (前身是 FhGFS) 是一个高性能的并行文件系统，使用分布式元数据架构，具备可伸缩和灵活性来满足业务的需求。BeeGFS 是免费的，提供专业的商业服务支持。
BeeGFS (前身是 FhGFS) 是一个领先的并行集群文件系统，开发的重点是性能，易于安装和管理。

标签: c++, 存储

https://git.beegfs.com/pub/v7

sonarQube

SonarQube是管理代码质量一个开放平台，可以快速的定位代码中潜在的或者明显的错误。

标签: java, cicd

https://www.sonarqube.org/

Drone

drone用go语言实现的，drone是前google员工2013年开源的一个CI/CD工具，现在已经拉了投资商业化了。企业版是付费的，我们用的是免费版本。drone更新比较快，之前稳定版本是0.7，今天部署的时候已经更新到0.8了。不过相关的文档不完善，很多都是coming soon。

标签: go, cicd

https://github.com/drone/drone

Handy

简洁易用的C++11网络库 / 支持单机千万并发连接

标签: c++

https://github.com/yedf/handy

Dive

用于探索docker镜像，图层内容以及发现缩小Docker图像大小的方法的工具

标签: golang

https://github.com/wagoodman/dive

Metis

织云Metis是聚焦在智能运维的应用实践集合，它基于腾讯已有的运维数据，将机器学习领域的分类、聚类、回归、降维等算法和运维场景相结合，旨在通过一系列基于机器学习的算法，对运维数据进行分析、决策，从而实现自动化运维的更高阶段。逐步降低对人指定规则的依赖，进而减少人为失误。

标签: python

https://github.com/Tencent/Metis
