ctrl-p

模糊搜索文件插件

安装

打开.vimrc加入下面内容后，执行:PluginInstall

使用

快捷键 功能
ctrl + p 开启文件搜索
F5 刷新当前目录内容
ctrl + f/b 切换显示模式
ctrl + d 只搜索文件名
ctrl + r 正则查找模式
ctrl + j/k 搜索结果中移动光标
ctrl + t 新标签页打开
ctrl + v/x 垂直分割打开 / 水平分割打开
ctrl + n/p 上下翻阅上次搜索历史
ctrl + y 在选中文件的父级目录创建新文件
ctrl + z 标记选择多个文件
ctrl + o 选择打开方式
